# @rag-forge/docs

Documentation site for RAG-Forge, built with Next.js 16 + Nextra v4.

## Development

```bash
pnpm --filter @rag-forge/docs dev
```

Open http://localhost:3000.

## Build

```bash
pnpm --filter @rag-forge/docs build
```

## Content

All MDX content lives in `content/`. Sidebar navigation is driven by `_meta.ts` files in each section.
